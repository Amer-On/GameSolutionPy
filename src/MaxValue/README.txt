Чтобы провести какие-либо тесты, чтобы проверить программу на правильность выполнения
достаточно просто закинуть входные данные в пустой input файл и поменять значение num в main.py
на номер input файла

Код не является полностью оптимизированным и может быть сильно упрощён, но не упрощён
для наглядности и потому что мне было лень


Оглавление:
-input  - директория с входными файлами
-output  - директория с выходными файлами
-python  - хранилище всех .py файлов

main.py  - основной код
outputting.py  - запись результатов в output файл
simple_input.py  - олимпиадное оформление с вводом через консоль
